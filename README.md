# clone-my-repo

> Clone my GitHub repository to `~/work/` directory

```shell
clone-my-repo
```

## Install
Download [`libexec/clone-my-repo`](libexec/clone-my-repo) to `~/.local/bin/`

## Install with [basher](https://github.com/basherpm/basher)
```
basher install vovanr/clone-my-repo
```
